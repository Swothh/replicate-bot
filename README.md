# ⚙️ Config
> Create a file named **.env** by copying the **.env.example** file. Write the **Discord bot token** and the **Replicate api key** to the file.

# 📜 Usage
> ***Make the adjustments mentioned above.***
>
> Install packages. \
> $ `npm install`
>
> Start bot. \
> $ `npm start`
